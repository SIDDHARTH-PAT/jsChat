<h1>jsChat</h1>
jsChat is a simple chat application built with Node.js, Express, and Socket.IO. It allows users to connect with each other in real-time and chat with each other.

<h2>Features</h2>
Real-time chat
User authentication
Chatroom creation and joining
Emojis support
